# chromebook-videospeedchange
Chromebookのギャラリーで、再生速度の変更が0.5ずつでムカついたので。
<br>
完全に自分で使う用です。需要はない。

## 使い方
1・ctrl + shift + jでコンソールを開く。
<br>
2・コードをはっつけてEnter (このとき変更したい速度を好きに変更する。デフォは±0.1)
<br>
3・+を押すと速度が0.1上がる。-を押すと速度が0.1下がる
